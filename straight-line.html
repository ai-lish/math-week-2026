📐 直線方程 — 面試示範教學

課題： 中四數學第3章 | 課件用途： 示範教學 / 面試演示

⏱️ 15分鐘面試示範教學流程

3 分鐘
引起動機 — 生活中既直線方程
提問：「你今日返學的路線，係唔係直線？」引出斜率概念。
展示三個生活例子（樓梯斜路 / 巴士收費 / MPF強積金）。
★ 教學技巧：設問引發思考，互動提問
2 分鐘
概念澄清 — 點與斜率缺一不可
展示：只有斜率 → 無限多條平行線；只有一點 → 無法確定。
結論：要確定一條特定直線，需要「斜率 + 一點」或「兩點」。
★ 教學技巧：反面論證，加深印象
8 分鐘
例題演繹 — 求直線方程三種方法
例題一（點 + 斜率）： 斜率3，通過(-5,2) → y = 3x + 17
例題二（點 + 點）： 通過(1,3)及(3,7) → y = 2x + 1
例題三（截距 + 截距）： x截距2，y截距6 → y = -3x + 6
★ 教學技巧：逐步拆解，每題後留白等學生回應
2 分鐘
課堂總結 — 一般式
將三個例題的答案統一化成 Ax + By + C = 0（A > 0）。
預告下節：點到直線的距離。
★ 教學技巧：首尾呼應，用例題結果作結

🌍 生活中與直線方程相關的概念

🚶 例1：斜路 / 樓梯的坡度
具體數值：輪椅通道標準坡度 1:12（約 4.8°）
概念：斜率 = 上升高度 / 水平距離
直線方程：y = (1/12)x — 每走12米水平距離，高度上升1米
🚌 例2：巴士 / 的士收費
具體數值：某巴士 $6 起步 + $1.2/公里
概念：y = mx + c（斜截式）
直線方程：y = 1.2x + 6 — x截距 = -5（無意義），y截距 = $6 起步價
💰 例3： MPF 強制性公積金
具體數值：每月固定 $1500 強積金
概念：正變關係（直線通過原點）
直線方程：y = 1500x — 斜率 = 每月底薪，y截距 = 0
📦 例4：物流運費
具體數值：首公斤 $24，每加1公斤 +$3
概念：兩點式 — 知道兩組數據就能建立方程
直線方程：y = 3x + 21 — 斜率 = $3/封閉，y截距 = $21 基本收費
⛽ 例5：汽車耗油量
具體數值：油缸60L，每行1公里耗0.2L
概念：負斜率（油量隨距離減少）
直線方程：G = 60 − 0.2x — 斜率 = −0.2，y截距 = 60L（出發時油量）
🏗️ 例6：安裝閉路電視費用
具體數值：基本安裝費 $12000 + 每部 $600
概念：y截距 = 固定成本，斜率 = 邊際成本
直線方程：C = 600n + 12000

面試加分技巧

✅ 用學生熟悉的例子（樓梯、收費）而非抽象概念
✅ 每個例子留 5-10 秒等學生思考
✅ 用顏色標記斜率（m）和截距（c）
✅ 總結時用一般式 Ax + By + C = 0 收尾

📝 例題一：點 + 斜率（點斜式）

y − y₁ = m(x − x₁)
📖 題目：已知一直線的斜率為 3，且通過點 (−5, 2)，求該直線的方程。
步驟：
① 代入 m = 3，x₁ = −5，y₁ = 2
  y − 2 = 3(x − (−5))
② 展開括號：y − 2 = 3(x + 5)
③ 展開：y − 2 = 3x + 15
④ 移項：y = 3x + 17

答案：y = 3x + 17

📝 例題二：點 + 點（兩點式）

y − y₁ = (y₂−y₁)/(x₂−x₁)(x − x₁)
📖 題目：求通過 A(1, 3) 及 B(3, 7) 兩點的直線方程。
步驟：
① 先求斜率：m = (7−3)/(3−1) = 4/2 = 2
② 代入 m = 2，選用 A點 (1,3)：
  y − 3 = 2(x − 1)
③ 展開：y − 3 = 2x − 2
④ 移項：y = 2x + 1

答案：y = 2x + 1

📝 例題三：截距 + 截距（截距式）

x/a + y/b = 1
📖 題目：已知一直線的 x 截距為 2，y 截距為 6，求該直線的方程。
步驟：
① 代入 a = 2，b = 6：
  x/2 + y/6 = 1
② 兩邊乘6：3x + y = 6
③ 移項：y = −3x + 6

答案：y = −3x + 6

統一成一般式 (Ax + By + C = 0, A > 0)

例題一：y = 3x + 17 → 3x − y + 17 = 0（× (−1)）
例題二：y = 2x + 1 → 2x − y + 1 = 0（× (−1)）
例題三：y = −3x + 6 → 3x + y − 6 = 0（× (−1)）

⚠️ 常見錯誤提醒

⚠️ 忘記處理負號： x − (−5) = x + 5，呢步最易錯
⚠️ 斜率公式符號： m = (y₂−y₁)/(x₂−x₁)，唔係 (x₂−x₁)/(y₂−y₁)
⚠️ 截距式 a、b 正負： 如果截距係負數，要小心代入
⚠️ 一般式 A>0： 化簡後記得調整符號，令 A 係正數

⭐ 核心要點（背誦版）

1️⃣ 斜率 m = 垂直變化 ÷ 水平變化
   (Δy/Δx)

2️⃣ 只有斜率 → 只能話明係邊組平行線
   （因為過任意一點都可以畫一條斜率3的線）

3️⃣ 斜截式 y = mx + c
   m = 斜率（傾斜程度），c = y截距（起點高度）

4️⃣ 一般式 Ax + By + C = 0（A > 0）
   所有直線都可以化成呢個形式
