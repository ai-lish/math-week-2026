## 1. 計算機大賽

### 1.1 遊戲概述
| 項目 | 內容 |
|------|------|
| **遊戲名稱** | 計算機大賽 |
| **形式** | 個人賽（個人鬥快） |
| **設備** | CASIO fx50FHII（學生自備或大會提供） |
| **時間** | 每局30秒-1分鐘（待定） |
| **核心能力** | 操作速度、按鈕熟悉度 |

### 1.2 遊戲目標
訓練學生熟練操作 CASIO fx50FHII 計算機，達到「肌肉記憶」境界，使解題時能專注於數學思維而非計算機操作。

### 1.3 遊戲規則

#### 1.3.1 比賽形式
- **參賽人數**：個人賽，每次1人上陣
- **答題方式**：看投影題目，快速使用計算機計算並顯示答案
- **評分標準**：速度優先，答案正確為前提

#### 1.3.2 題目類型
1. **基礎運算**：加、減、乘、除
2. **百分比計算**：找出百分比、折扣後價格
3. **M+/M- 記憶運算**：連續計算多個數值後求總和
4. **倒數與平方根**：快速計算倒數、平方根
5. **括號運用**：複雜運算式需要正確使用括號
6. **代碼輸入**：特定計算機功能鍵組合

#### 1.3.3 比賽流程
```
1. 主持宣佈題目类型
2. 投影顯示題目
3. 參賽者開始計算
4. 主持人計時
5. 參賽者完成後舉手/按鐘
6. 核對答案
7. 正確者得分
```

#### 1.3.4 計分方式
| 題目難度 | 正確答題分數 |
|---------|-------------|
| 基礎題 | +1 分 |
| 進階題 | +2 分 |
| 挑戰題 | +3 分 |

### 1.4 深層教育意義
- **肌肉記憶**：將常用操作自動化，釋放認知資源
- **工具掌握**：深入了解計算機功能，不被工具局限
- **效率意識**：發現更快捷的按鍵順序

### 1.5 設備需求
- [ ] CASIO fx50FHII 計算機 × 數量（根據參賽人數）
- [ ] 投影機 + 屏幕
- [ ] 計時器（可用電話或電腦）
- [ ] 計分表

### 1.6 注意事項
⚠️ 計算機型號必須為 CASIO fx50FHII（統一規格）  
⚠️ 禁止使用任何應用程式或電話計算機  
⚠️ 答案須完全正確方可得獎

## 2. 公式圖像大配對

### 2.1 遊戲概述
| 項目 | 內容 |
|------|------|
| **遊戲名稱** | 公式圖像大配對 |
| **形式** | 個人/團隊限時卡牌配對 |
| **設備** | 實體卡牌或電子配對系統 |
| **時間** | 每局30秒-2分鐘（根據難度） |
| **核心能力** | 數學直覺、公式與圖像連結 |

### 2.2 遊戲目標
讓學生建立「公式 ↔ 圖像」的雙向連結，從視覺層面理解代數表達式的幾何意義，培養數學直覺。

### 2.3 遊戲規則

#### 2.3.1 基本形式
- **卡牌類型**：一側為「公式/代數」，另一側為「圖像/表格」
- **配對目標**：在限時內找出所有匹配的公式-圖像配對

#### 2.3.2 難度級別
| 難度 | 網格大小 | 卡牌數量 | 時間限制 |
|------|---------|---------|---------|
| 初級 | 2×2 | 2對 | 30秒 |
| 中級 | 4×4 | 8對 | 60秒 |
| 高級 | 6×6 | 18對 | 90秒 |
| 專家 | 8×8 | 32對 | 120秒 |

#### 2.3.3 配對類型
1. **代數 ↔ 圖形**：如 `a²-b²` ↔ 兩個正方形面積差示意圖
2. **公式 ↔ 表格**：如 `V=IR` ↔ 電壓-電流-電阻表格
3. **公式 ↔ 坐標圖**：如 `y=mx+c` ↔ 直線圖像

#### 2.3.4 題庫設計
- **每年級一套**：最少六套（SSS1-3, HKCSE 1-3）
- **可混合使用**：可根據需要混配不同年級題目
- **類別分類**：代數、幾何、三角、統計、函數

#### 2.3.5 計分方式
| 完成時間 | 分數 |
|---------|------|
| 規定時間內完成 | +10 分 |
| 縮短50%時間 | +15 分（Bonus） |
| 完全零錯誤 | +5 分（Bonus） |

### 2.4 深層教育意義
- **從記憶到理解**：公式不再是抽象符號，而是具體圖像
- **雙向思維**：看到公式想到圖形，看到圖形想到公式
- **數學直覺**：培養「看一眼就知道」的直覺能力

### 2.5 設備需求
- [ ] 配對卡牌套裝（每套包含所有難度）
- [ ] 計時器
- [ ] 計分表
- [ ] 答案核對卡

### 2.6 注意事項
⚠️ 卡牌需正反分明，方便識別  
⚠️ 可使用電子版本（iPad）提高效率  
⚠️ 混合年級使用時需清楚標示題目來源

## 3. 四則運算拉力賽

### 3.1 遊戲概述
| 項目 | 內容 |
|------|------|
| **遊戲名稱** | 四則運算拉力賽 |
| **形式** | 雙人電子拔河對決（iPad） |
| **設備** | iPad（展示用）+ 計算紙（學生手寫） |
| **時間** | 每局30秒-1分鐘 |
| **核心能力** | 心算能力、專注力 |

### 3.2 遊戲目標
訓練學生脫離計算機，依賴心算完成四則運算，強化數字敏感度與認知靈活性。

### 3.3 遊戲規則

#### 3.3.1 基本形式
- **人數**：iPad左右兩側各站立一人
- **答題方式**：同步輸入，鬥快且正確
- **禁止使用**：計算機（全程手寫心算）

#### 3.3.2 難度遞進系統
```
第一關：基礎四則
   ↓ 雙方各勝一局後升級
第二關：乘除混合（左至右運算）
   ↓ 雙方各勝一局後升級
第三關：先乘除後加減
   ↓ 雙方各勝一局後升級
第四關：括號優先運算
```

#### 3.3.3 題目範例
| 關卡 | 範例題目 |
|------|---------|
| 第一關 | `7 + 5 = ?`、`24 ÷ 6 = ?` |
| 第二關 | `6 × 4 ÷ 3 = ?`、`48 ÷ 8 × 2 = ?` |
| 第三關 | `3 + 6 × 4 = ?`、`48 - 12 ÷ 3 = ?` |
| 第四關 | `(3 + 5) × 2 = ?`、`(24 ÷ (2 + 6)) × 3 = ?` |

#### 3.3.4 拔河機制
- **電子拔河**：iPad顯示拉力條，雙方答案正確時拉向己方
- **勝出條件**：將拉力標誌拉過己方底線
- **難度調整**：越接近勝利，題目難度越高

#### 3.3.5 計分方式
| 結果 | 分數 |
|------|------|
| 勝出一局 | +10 分 |
| 連勝Bonus | +5 分（每多勝一局額外） |
| 失敗方（已完成題目）| +2 分（參與分） |

### 3.4 深層教育意義
- **脫離工具依賴**：心算是數學思维的基礎
- **專注力訓練**：限時高壓環境培養專注
- **面對壓力**：在劣勢中保持冷靜
- **數字敏感度**：提高對數字的直覺與敏銳度

### 3.5 設備需求
- [ ] iPad × 1（展示用）
- [ ] 計算紙（每位參賽者）
- [ ] 計時器
- [ ] 計分表

### 3.6 注意事項
⚠️ 必須使用計算紙手寫答案，不可使用計算機  
⚠️ 答案需完全正確方可視為勝出  
⚠️ 主持需即時核對答案並更新拉力條
