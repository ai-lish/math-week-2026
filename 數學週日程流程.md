## ⏰ 每日時間流程（30分鐘）

### 標準遊戲日流程
```
10:25 - 10:30  準備時間（設備就位、參賽者就位）
10:30 - 10:32  主持宣佈今日遊戲規則（2分鐘）
10:32 - 10:35  示範題目（3分鐘）
10:35 - 10:55  正式比賽（20分鐘）
10:55 - 11:00  統計積分、公佈今日積分榜（5分鐘）
```

### 總結日流程
```
10:25 - 10:30  準備時間
10:30 - 10:45  小測試（15分鐘）
10:45 - 11:00  積分統計 + 本週總結（15分鐘）
```

### 決賽日流程
```
10:25 - 10:30  準備時間
10:30 - 10:35  主持介紹決賽規則（5分鐘）
10:35 - 10:50  三項遊戲決賽（每項5分鐘，共15分鐘）
10:50 - 11:00  評審環節 + 即場積分統計（10分鐘）
```
