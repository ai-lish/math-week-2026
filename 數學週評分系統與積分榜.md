## 1. 遊戲計分方式

### 1.1 計算機大賽

| 項目 | 內容 |
|------|------|
| **設備** | CASIO fx50FHII（午膳時間排隊）|
| **遊戲時長** | 30秒/次 |
| **難度** | 基礎運算 → 記憶運算 → 綜合應用 |

#### 計分規則
| 結果 | 分數 |
|------|------|
| 答對 | **+1 分** |
| 答錯 | **-1 分**（即時顯示正確答案）|

#### 龍虎榜排名方式
- 按**遊戲總分**排名
- 分數相同時，按**遊玩次數**少的優先

---

### 1.2 公式圖像配對

| 項目 | 內容 |
|------|------|
| **設備** | iPad（午膳時間排隊）|
| **難度選項** | 4×4、6×6、8×8 |

#### 計分規則
| 難度 | 完成分數 |
|------|---------|
| 4×4（8對）| **1 分** |
| 6×6（18對）| **2 分** |
| 8×8（32對）| **3 分** |

| 結果 | 分數 |
|------|------|
| 限時完成 | 按難度得分 |
| 超時 | **0 分** |

#### 龍虎榜排名方式
- 按**完成時間**排名（快者優先）
- 分數相同時，按**最佳時間**排名

---

### 1.3 四則運算拉力賽

| 項目 | 內容 |
|------|------|
| **設備** | iPad（雙人對決）|
| **人數** | 2人同時排隊 |
| **難度** | 基礎 → 乘除混合 → 先乘除後加減 → 括號優先 |

#### 計分規則
| 結果 | 分數 |
|------|------|
| 完勝（提前拉過底線）| **2 分** |
| 限時勝出（時間到最近底線）| **1 分** |
| 失敗 | **0 分** |

#### 龍虎榜排名方式
- 按**遊戲總分**排名
- 分數相同時，按**勝率**高的優先

## 2. 數學積分計算

### 2.1 積分兌換

```
1 遊戲分 = 1 數學積分
```

**直接兌換，冇轉換表！**

| 遊戲分 | 數學積分 |
|--------|---------|
| 1 分 | 1 分 |
| 5 分 | 5 分 |
| 10 分 | 10 分 |
| 15 分 | 15 分 |

### 2.2 數學積分用途
- 數學週結束後可用於兌換獎勵
- 具體獎勵內容待定
- 兌換規則另行公佈

## 3. 龍虎榜設計

### 3.1 顯示位置
| 位置 | 設備 |
|------|------|
| iPad 屏幕 | 每次遊戲後即時顯示 |
| 投影 | 午膳時間持續投影 |
| 網頁 | math-week-2026 網站 |

### 3.2 龍虎榜類型
三個遊戲 **分開排名**，各有一個獨立龍虎榜：
- 📱 計算機大賽 龍虎榜
- 🧩 公式圖像配對 龍虎榜
- 🔢 四則運算拉力賽 龍虎榜

#### 顯示欄位
| 欄位 | 說明 |
|------|------|
| #排名 | 當前排名 |
| 姓名 | 學生姓名 |
| 班別 | 班級 |
| ██████ 分數 | 視覺化積分條 |
| 總分 | 遊戲總分 |
| 次 | 遊玩總次數 |
| ↗↘ | 排名變化（較上日）|
| ⭐ | 最佳時間（拉力賽）|

---

### 3.4 數據記錄（iPad App）

每次遊戲後記錄：
```
1. 學生姓名
2. 班別
3. 遊戲分數
4. 完成時間（適用於公式配對）
5. 遊戲時間戳
```

#### 數據流向
```
iPad輸入 → 即時更新龍虎榜 → 同步到Google Sheets → 網頁顯示
```

## 4. 獎勵制度

### 4.1 獎勵來源
- 獎勵以**數學積分**兌換
- 數學積分 = 遊戲分（1:1兌換）

### 4.2 獎勵等級（待定）
| 等級 | 積分要求 | 獎勵內容 |
|------|---------|---------|
| 🥉 銅牌 | 10+ 分 | 待定 |
| 🥈 銀牌 | 30+ 分 | 待定 |
| 🥇 金牌 | 50+ 分 | 待定 |
| 💎 鑽石 | 100+ 分 | 待定 |

### 4.3 頒發安排（待定）
- 數學週期間：每日積分統計
- 閉幕日：統一兌換獎勵
